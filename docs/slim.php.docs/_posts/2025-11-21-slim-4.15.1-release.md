We are pleased to release Slim 4.15.1 today, with official support for [PHP 8.5](https://www.php.net/releases/8.5/en.php) which was released yesterday. 

Slim 4 is a mature project and so we are keen to ensure that it remains stable for the projects that it underpins. To that end, we also upgraded our internal testing so that we can use the latest versions of PHPUnit.

The full changelog can be seen [here](https://github.com/slimphp/Slim/releases/tag/4.15.1).
